# DirectX11　ゲーム作品

DirectX11で作成した個人制作アクションゲームです。  
重力方向を操作できる「重力ゾーン」を用いた高速移動アクションをテーマに開発しました。

## 制作情報

- **制作期間**：約8ヶ月  
  - 前半4ヶ月：エンジン基礎構築  
  - 後半4ヶ月：独自描画機能の拡張・実装
- **開発人数**：1人

## 作品概要

リアルタイムで重力ゾーンを配置し、重力方向を切り替えながら高速移動するアクション作品です。  
プレイ体験の強化を目的に、ゲーム性だけでなく描画・演出面も重視して実装しました。

## アピールポイント

- Weighted Blended OITによる半透明表現の安定化
- ディファードレンダリングによる多数ライト処理
- レイマーチングによる雲海表現（LOD・早期終了対応）
- GPUパーティクルによる速度感の可視化
- カスケードシャドウマップによる広域対応の影表現
- MSDFによる3D空間上でのテキスト描画
- クォータニオンによるMSDFの回転制御
- 各処理を統合した状態で **100fps以上** の安定動作
