DirectX11で作成したアクションゲーム

【制作期間】
約8ヶ月（前半4ヶ月でエンジンの基礎構築、後半4ヶ月で独自の描画機能の拡張・実装）
開発人数：1人

【作品概要】
重力方向を操作できる重力ゾーンを用いた高速移動アクションの作品です。
リアルタイムでのゾーン配置と高速移動を軸に、描画・演出の両面から体験の強化を目指しました。

【アピールポイント】
・Weighted Blended OITによる半透明表現の安定化
・ディファードレンダリングによる多数ライト処理
・レイマーチングによる雲海表現（LOD・早期終了対応）
・GPUパーティクルによる速度感の可視化
・カスケードシャドウマップによる広域に対応した影
・MSDFによる3D空間上でのテキスト描画
・クォータニオンによるMSDFの回転制御
・各処理を統合した状態で100fps以上の安定動作


【使用技術】
C++ / DirectX11

【使用ライブラリ】
・DirectX11
・ImGui
・Assimp
